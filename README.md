# ivantitov.tech

Персональный сайт-портфолио разработчика 1С, посвящённый концепции «вайбкодинга» и продуктам для автоматизации 1С.

## Обзор

Сайт представляет собой современную одностраничную landing page с информацией о разработчике, его продуктах и блоге. Особенности дизайна:

- Неоновый градиентный стиль с glow-эффектами
- Анимации при скролле (scroll reveal)
- Печатная машинка для демонстрации кода
- Адаптивная мобильная навигация
- SEO-оптимизация (Schema.org, мета-теги)

## Структура проекта

```
site/
├── index.html          # Главная страница
├── assets/             # Статические ресурсы (favicon, изображения)
├── css/                # Стили
│   ├── variables.css   # CSS-переменные (цвета, шрифты)
│   ├── base.css        # Базовые стили и сброс
│   ├── layout.css      # Сетка, навигация, контейнер
│   ├── sections.css    # Стили секций (hero, about, products, blog, contacts)
│   └── animations.css  # Анимации и переходы
└── js/                 # JavaScript
    ├── main.js         # Основная логика (навигация, мобильное меню)
    ├── typing.js       # Эффект печатной машинки
    └── scroll-reveal.js # Анимации при скролле
```

## Секции сайта

1. **Hero** — главный экран с заголовком и демонстрацией кода
2. **О вайбкодинге** — описание философии разработки
3. **Продукты** — карточки продуктов (МультиВыгрузка 1С, АвтоПоток, КодМастер)
4. **Блог** — список статей
5. **Контакты** — форма обратной связи и ссылки на соцсети

## Технологии

- Чистый HTML5, CSS3, JavaScript (без фреймворков)
- Google Fonts (Unbounded, Commissioner, JetBrains Mono)
- SVG-иконки

## Локальный запуск

Любой статический сервер:

```bash
# Python
python -m http.server 8000

# Node.js (npx)
npx serve

# Или открыть index.html напрямую в браузере
```

## Разработка

Для изменения содержимого редактируйте `index.html`. Стили и скрипты модульны — каждая секция CSS отвечает за конкретную часть интерфейса.
